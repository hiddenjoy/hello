웹프로그래밍개론

  • 수업내용

    HTML은 웹페이지를 구성하는 마크업 언어로, 태그를 사용하여 구조화된 문서를 작성합니다. 웹 브라우저는 HTML 문서를 해석하여 사용자에게 시각적인 화면을 제공합니다. HTML 문서는 head와 body로 구성되며, head는 문서의 메타데이터와 스타일시트를 포함하고, body는 실제 컨텐츠를 포함합니다. 각각의 요소는 태그로 구분되며, 태그는 속성을 포함할 수 있습니다. HTML5에서는 시멘틱 요소를 사용하여 웹페이지를 더 의미론적으로 작성할 수 있습니다. 이러한 HTML 기초 개념과 함께, 웹페이지를 만들기 위해 필요한 여러 태그와 속성, 그리고 웹페이지의 구조화와 스타일링 방법 등이 다루어집니다.

  • 과제안내

    웹페이지 구현 과제: HTML과 CSS를 사용하여 개인 포트폴리오 웹페이지를 만들고, GitHub Pages를 이용해 배포하기.

디지털 미디어의 이해

  • 수업내용

    인터넷은 컴퓨터 네트워크들이 연결되어 정보를 공유하는 전 세계적인 네트워크입니다. 인터넷은 웹, 이메일, 파일 전송, 채팅 등 다양한 통신 방식을 지원하며, HTTP, TCP/IP, DNS 등의 프로토콜을 사용하여 동작합니다. 인터넷은 또한 보안과 개인 정보 보호에 대한 이슈도 다루어지며, 이러한 문제를 해결하기 위한 보안 기술도 중요합니다.

  • 과제안내

    인터넷을 활용하여 자신이 관심 있는 주제에 대한 정보를 수집하고, 그 정보를 기반으로 직접 만든 블로그 글을 작성해보세요. 작성한 글을 인터넷을 통해 공유하는 방법에 대해서도 생각해보세요.

디지털 에쓰노그라피

  • 수업내용

    전통 에쓰노그라피와 디지털 에쓰노그라피의 차이는 전자적 장비를 사용하는가 여부에 있습니다. 전통적인 에쓰노그라피는 종이와 붓을 사용하여 손으로 직접 쓰는 방식이지만, 디지털 에쓰노그라피는 디지털 펜과 태블릿 등의 전자적 장비를 사용하여 작업하는 방식입니다. 디지털 에쓰노그라피는 전통 에쓰노그라피에 비해 수정과 저장, 전송 등이 용이하며, 디지털 매체에서 사용하기 적합한 이미지를 만들 수 있습니다.

  • 과제안내

    전통적인 에쓰노그라피와 디지털 에쓰노그라피를 비교 분석해보고, 자신이 선호하는 방식으로 에쓰노그라피 작업을 하여, 해당 작업을 온라인에 공유하는 방법에 대해 생각해보세요.
